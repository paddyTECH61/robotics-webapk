# Robotics Web App

A web application that explores the **impact of robots on our daily lives**.  
This project demonstrates how robots **positively contribute** to society while also highlighting potential challenges, such as automation replacing certain jobs.

---

## Purpose

The app aims to:

- Educate users about the **benefits of robotics** in healthcare, manufacturing, and daily life.  
- Show real-world examples of robots **improving efficiency and safety**.  
- Encourage discussion about **automation and its effects on employment**.  

---

## Features

- Interactive web pages showcasing different types of robots  
- Informative sections on **robotic applications** in various industries  
- Balanced view highlighting **both opportunities and risks** of robotics  
- Responsive design for desktop and mobile  

---
